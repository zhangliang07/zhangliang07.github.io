A program for interpolation on 3D surfaces
2015/9 – 2015/10

For 3D surfaces constructed by discrete points, there are requirements to get the position of a specific point on the surface. The Program imitates the corresponding function in Matlab called “scatteredInterpolant”.

After importing the scattered points from text files, firstly it generates Delaunay Triangles by splitting triangles constantly. Secondly, it finds the triangle whose distance between the given point is closest. Thirdly it calculates the projected point on the plane triangle. At last, it returns the position of the projected point, which is the position of the given point. It was developed by C++ independently.
